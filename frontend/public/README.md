# Dossier Public - Assets Statiques

Ce dossier contient les fichiers statiques qui seront servis à la racine de l'application.

## Fichiers requis pour une PWA complète

### Favicons
- `favicon.svg` - Favicon moderne (recommandé)
- `favicon.png` - Favicon de fallback
- `apple-touch-icon.png` - Icône pour iOS (180x180px)

### Icons PWA
- `icon-192.png` - Icône 192x192px pour PWA
- `icon-512.png` - Icône 512x512px pour PWA

### Images
- `og-image.png` - Image pour les partages sociaux (1200x630px recommandé)

## Note
Ces fichiers peuvent être générés en ligne avec des outils comme :
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
