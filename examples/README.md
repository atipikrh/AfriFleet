# Exemples et Prototypes

Ce dossier contient des versions alternatives ou des prototypes de l'application AfriFleet.

## Fichiers

### `index-standalone.html`
Version standalone de l'application avec JavaScript vanilla (sans React).
- **Statut** : Prototype/Exemple
- **Usage** : Peut être ouvert directement dans un navigateur
- **Note** : Cette version n'est plus maintenue. L'application principale utilise React et se trouve dans `frontend/`

## Application Principale

L'application principale se trouve dans le dossier `frontend/` et utilise :
- React + TypeScript
- Vite pour le build
- Tailwind CSS
- Architecture moderne avec composants

Pour démarrer l'application principale :
```bash
npm run dev
```
